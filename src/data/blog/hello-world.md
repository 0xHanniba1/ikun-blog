这是新博客的第一篇文章。

## 为什么开新站

之前的博客 [0xhanniba1.github.io](https://0xhanniba1.github.io/) 越来越像生活碎碎念 —— 周记、年度记录、随手的想法。

我想把**技术内容单独拎出来**，让两边都更纯粹：

- **碎碎念站**：周记、年度记录、生活想法
- **技术站**（也就是这里）：工程实践、AI agent、踩坑笔记 与思考

## 这个站怎么搭的

| 组件 | 选型 |
|---|---|
| 框架 | [Astro](https://astro.build/) v5 |
| 主题 | [AstroPaper](https://github.com/satnaing/astro-paper) |
| 部署 | Cloudflare Pages |
| 域名 | Cloudflare 注册 |
| 内容 | Markdown |
| 评论 / 统计 | 暂未接入 |

整套加起来 **0 服务器成本**，只花了一个域名的钱。后面会单独写一篇详细的搭建过程。

## 接下来打算写什么

- AI 编程相关的工程实践（Claude Code、Cursor、各种 agent 框架）
- 测试 / 自动化相关的踩坑
- 零散的工具链笔记（Git、shell、效率工具）
- 可能会有一些读源码的笔记
